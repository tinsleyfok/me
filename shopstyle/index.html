ShopStyle Redesign
A redesign to create a novel experience for searching items

BRIEF

ShopStyle is the world’s leading fashion and lifestyle shopping platform for influencers and consumers to search, explore, purchase and get cash back - all in one place. As a loyal user of Shopstyle App and an passionate UX designer, I found some existing problems with the App. So I decided to add a new feature and redesign the App.

ROLE

UX Research
UX/UI Design

OVERVIEW

Back in 2007, ShopStyle was described as "a search engine devoted to fashion" when it was launch. By the end of 2015 ShopStyle’s global sites generated five million searches per week and listed 12 million products from 1,400 retailers. After being acquired by Rakuten in 2017, there is also a new feature-cash back. There is a spectrum of user patterns, from niche users who use ShopStyle to browse the latest fashion trend, to regular users who use the App as a shortcut to search favorite items and easily compare prices.


OBJECTIVE

However, there is no easy option for users to search for items without knowing the brand's name. This means that as long as users don't know the name of an item, there is a high probability that they won't be able to find and buy it. The objective is to add a new feature - search by image - to fill in the gap.

If ShopStyle can access all the images of the fashion products, filled by machine learning, users can search by images and do not need to know brand name to find the item.

USER PAIN POINT

The following are the core problems this redesign aims to solve:

No easy option to search without knowing brand name - Some users find clothes they like from magazines or the Internet, but they don't know the brand but want to buy them. They can't find the answer in this "Google of fashion."

Viewing cashback history is unorganized and difficult to navigate - Cashback is a big new feature of ShopStyle. It needs to redirecr to Rakuten's App to view it, which discourages users to use this feature.
